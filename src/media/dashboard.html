Level 0 ✓
Interactive
Strict security. Prompts user for manual validation on every command.
Level 1 ✓
Read-Only YOLO
Auto-executes safe, non-destructive discovery commands (ls, cat, grep).
Level 2 ✓
Scoped YOLO
Fine-grained control. Allows custom categories of execution toggles.
Level 3 ✓
Full YOLO
Unrestricted execution. Fully autonomous (subject to hard blacklist).
